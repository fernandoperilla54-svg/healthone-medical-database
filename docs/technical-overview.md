# HealthOne - Documentación Técnica

## Descripción del Sistema
Sistema de base de datos para gestión médica y seguros.

## Entidades Principales
- Pacientes
- Médicos 
- Hospitales
- Seguros
- Consultas
- Medicamentos
- Prescripciones

## Esquema de Base de Datos
- 12 tablas normalizadas
- Relaciones complejas
- Integridad referencial
