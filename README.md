# HealthOne Medical Database System

## Sistema Integral de Gestión Médica y Seguros

### Descripción del Proyecto
Sistema de base de datos diseñado para compañías de seguros médicos que permite el seguimiento completo de pacientes, proveedores, consultas médicas, medicamentos y reclamaciones.

### Características Principales
- Gestión de Pacientes: Información demográfica, seguros, historial médico
- Control de Proveedores: Médicos, especialidades, afiliaciones hospitalarias  
- Seguimiento de Medicamentos: Prescripciones, interacciones, efectos secundarios
- Análisis de Reclamaciones: Procesamiento y seguimiento de claims médicos
- Sistema de Consultas: Tipos de consulta (nuevas, seguimiento, rutina)

### Estructura de la Base de Datos
- 12 entidades principales interrelacionadas
- Normalización 3FN aplicada
- Arquitectura con supertipos/subtipos para consultas
- Relaciones recursivas para seguros familiares

### Tecnologías Utilizadas
- PostgreSQL
- SQL Avanzado
- Python (ETL y automatización)
- Power BI (Dashboards y reporting)
